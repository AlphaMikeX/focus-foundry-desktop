# Focus Foundry

Electron + React + TypeScript desktop app for focus sessions using privacy-safe generic input signals.

## Node version

This project is configured to work cleanly on Node `18.16+` (including Node `18.16.0`) without ESLint engine warnings.

## Run in dev

```bash
npm install
npm run dev
```

## Build app

```bash
npm run dist
```

Windows installer artifact is emitted in `dist/` (NSIS target).
